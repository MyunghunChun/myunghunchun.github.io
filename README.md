```markdown
# Portfolio (static)

이 저장소는 GitHub Pages로 배포 가능한 정적 포트폴리오 템플릿입니다.
아래 단계로 배포하세요.

## 포함된 파일
- index.html
- assets/css/style.css
- assets/js/main.js
- blog/index.html

## 배포 방법 (간단)
1. 새 리포지토리를 생성하고 모든 파일을 루트에 추가합니다.
2. 커밋 후 GitHub에 푸시합니다.
3. GitHub 저장소의 Settings → Pages에서 배포 브랜치를 `main` (또는 `gh-pages`) 의 루트로 설정하세요.
   - repo 이름이 `username.github.io` 라면 브랜치 `main`의 루트를 사용하면 됩니다.
4. 잠시 기다리면 사이트가 https://<your-username>.github.io/ 에서 보입니다.

## 수정 안내
- 이름/이메일/프로젝트/경력 등은 `index.html`에서 직접 수정하세요.
- 더 많은 블로그 글을 원하면 `blog/` 아래에 정적 HTML 페이지를 추가하거나 Jekyll을 도입해 `_posts`를 사용하세요.

## 도움 필요하면
원하시면 이 파일들을 Jekyll 형식(동적 포스트 빌드)으로 바꾸거나, Netlify/Hugo 템플릿으로 변환해 드립니다.
```
